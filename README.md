# Pran Website (Vite + React)

Landing page for Pran — spectroscopy-enhanced waste classification for cleaner recycling streams.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Replace copy / numbers
Most text and metrics live in `src/content.js`.
Swap in your latest assumptions, quotes, or URLs there.
